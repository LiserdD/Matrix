Десятичные дроби: вводятся через точку или запятую в зависимости от настроек вашей операционной системы. Если вы ввели десятичную дробь и программа выдала сообщение "Можно вводить только числа!" попробуйте заменить разделительный знак "." на "," или наоборот.
Например: 1.2; -0.4 (или 1,2; -0,4)
Дробные числа (-): вводятся через "/".
Например: 1/2; -3/4 ; 8/3.
Символы (-): Скобки, буквы, операции сложения, вычитания, умножения, возведения в степень и прочее не поддерживаются!
Некоректно заполненые поля подсвечиваются красным цветом.
Сложение и вычитание чисел. Теория:
Складывать и вычитать можно матрицы одного размера в результате получается матрица того же размера.
Определение. Сложение матриц (сумма матриц): A + B есть операция вычисления матрицы C, все элементы которой равны попарной сумме всех соответствующих элементов матриц A и B, то есть каждый элемент матрицы C равен:

сij = aij + bij

Определение. Вычитание матриц (разность матриц): A - B есть операция вычисления матрицы C, все элементы которой равны попарной разности всех соответствующих элементов матриц A и B, то есть каждый элемент матрицы C равен:

сij = aij - bij
