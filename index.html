Правила ввода чисел:
Целые числа:
Например: 2; 0; -6.
Десятичные дроби: вводятся через точку или запятую в зависимости от настроек вашей операционной системы. Если вы ввели десятичную дробь и программа выдала сообщение "Можно вводить только числа!" попробуйте заменить разделительный знак "." на "," или наоборот.
Например: 1.2; -0.4 (или 1,2; -0,4)
Дробные числа (-): вводятся через "/".
Например: 1/2; -3/4 ; 8/3.
Символы (-): Скобки, буквы, операции сложения, вычитания, умножения, возведения в степень и прочее не поддерживаются!
Некоректно заполненые поля подсвечиваются красным цветом.
Умножение матриц. Теория:

Определение: Результатом умножения матрицы Am×n на матрицу Bn×k будет матрица Cm×k такая, что элемент матрицы C, стоящий в i-той строке и j-том столбце (cij), равен сумме произведений элементов i-той строки матрицы A на соответствующие элементы j-того столбца матрицы B.

Замечание: Две матрицы можно перемножить если количество столбцов первой матрицы равно количеству строк второй матрицы.

Пример 1: Найти матрицу C равную произведению матриц:
A =
42
90
и B =
31
-34
Решение:

C = A · B =

42
90
·
31
-34
=
612
279

Элементы матрицы C вычисляются следующим образом:

c11 = a11·b11 + a12·b21 = 4·3 + 2·(-3) = 12 - 6 = 6

c12 = a11·b12 + a12·b22 = 4·1 + 2·4 = 4 + 8 = 12

c21 = a21·b11 + a22·b21 = 9·3 + 0·(-3) = 27 + 0 = 27

c22 = a21·b12 + a22·b22 = 9·1 + 0·4 = 9 + 0 = 9
